core.repository.OpenDSURepository.<T>

Provide the Base implementation to a global single OpenDSU Repository implementation capable of, via the decorated properties of DSUModels, handle all of the OpenDSU API related to CRUD operations in a single, scalable, maintainable and declarative fashion, supporting:

  • Automatic CRUD operations, just by updating its corresponding DSUModel instance and running it through the appropriate OpenDSURepository;
  • Automatic validations && easily extendable for added validations;
  • Controlled accesses: Ability easily to add business logic at key points of any CRUD operations
.
