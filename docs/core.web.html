# static exports.fromWeb(appOrUrl, slot, mountPathopt, mountOptionsopt)

Depending on whether and url or an app name is supplied:

  • App Name: Communicates with ApiHub to get seeds from SSApps installed in that ApiHub instance:
    • slot 'primary': gets the seed file from the App's 'wallet-patch' folder;
    • slot 'secondary' gets the seed file from the App's Constan

Parameters:
Name Type Attributes Description
appOrUrl string

the app name to look for in ApiHub or the URL to an endpoint to call
slot "primary" | "secondary" | undefined

the slot where the seed can be found. 'primary' refers to the 'wallet slot' and secondary to the 'apps slot'. defaults to 'primary' and is mandatory when an App Name is provided. is discarded if a URL is detected in the appName
mountPath string <optional>

defines the mount path. defaults to the property key
mountOptions DSUIOOptions <optional>

options to be passed to OpenDSU for the mounting operation
