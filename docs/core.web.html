# static fromWeb(appOrUrl, slot, triggerMountopt, deriveopt, mountPathopt, mountOptionsopt, argsopt)

Depending on whether and url or an app name is supplied:

  • App Name: Communicates with ApiHub to get seeds from SSApps installed in that ApiHub instance:
    • slot 'primary': gets the seed file from the App's 'wallet-patch' folder;
    • slot 'secondary' gets the seed file from the App's ConstantsApi#APP_FOLDER

Retrieves the Seed during the DSUOperation.CREATION operation and DBOperations.CREATE phases and stores in the in object under the matching property key

calls mount on this property to mount the seed during the DSUOperation.EDITING operation;

Parameters:
Name Type Attributes Description
appOrUrl string

the app name to look for in ApiHub or the URL to an endpoint to call
slot "primary" | "secondary" | undefined

the slot where the seed can be found. 'primary' refers to the 'wallet slot' and secondary to the 'apps slot'. defaults to 'primary' and is mandatory when an App Name is provided. is discarded if a URL is detected in the appName
triggerMount boolean <optional>

Decides if the DSU retrieved will be mounted or not. defaults to true. all following params require this to be true to be used
derive boolean | number <optional>

defines how many (if any) times the KeySSI will be Derived before use (only used when {@param triggerMount} is true)
mountPath string <optional>

defines the mount path. defaults to the property key
mountOptions DSUIOOptions <optional>

options to be passed to OpenDSU for the mounting operation
args Array.<any> <optional>

Not used in current implementation. Meant for extending decorators
